# Directus Mobile app for ios/android

Transform your content management experience with Directus, the powerful open data platform now available on mobile. Access, manage, and control your content with ease, right from your smartphone or tablet. Add widgets to your homescreen displaying your collections and receive Push Notifications on data updates.

App status: Production

iOS: https://apps.apple.com/app/directus-mobile/id6740696350

Android: https://play.google.com/store/apps/details?id=com.martijnmichel.directusexpo.app

### Supported interfaces

**Inputs**
- Text
- TextArea
- Wysiwyg
- Number
- Code (json)

**Selections**
- Toggle
- Datetime
- Repeater
- Color
- Dropdown
- Checkboxes
- Dropdown Multiple
- Radio Buttons

**Relational**
- File
- Image
- Files
- Builder (M2A)
- Many to Many
- One to Many
- Many to One
- Translations

**Presentation**
- Dividers
- Buttons
- Notices

**Groups**
- Accordion
- Detail Group
- Raw Group

**Other**
- Collection Item
- Hash
- Slider
